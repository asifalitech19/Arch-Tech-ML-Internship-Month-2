## 🚀 Projects Overview

### 🏡 Task 3: Housing Price Prediction 
A regression project utilizing the California housing dataset to build a machine learning model to predict house prices based on features like location, number of rooms, and population[ 8].

**Key Objectives:**
* Clean the data to ensure high-quality inputs.
* Select important features that drive housing prices.
* Train the model using advanced regression techniques.
* Evaluate its accuracy using MSE and R-squared metrics.

### 🌸 Task 4: Iris Flower Classification 
A multi-class classification project using the famous Iris dataset to build a machine learning model to classify iris flowers into their species based on features like petal length, petal width, sepal length, and sepal width.

**Key Objectives:**
* Preprocess the data for optimal model performance.
* Train the model to accurately distinguish between species.
* Evaluate its accuracy using classification reports and accuracy scores.
